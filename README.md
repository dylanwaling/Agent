# Document Q&A Pipeline

A clean, robust document processing and question-answering system using **Docling → LangChain → Flask**.

## 🚀 Quick Start

```bash
# Install dependencies
pip install -r docs/requirements.txt

# Run the web app
python web_app.py

# Or check system components
python system_check.py
```

```

## 📁 File Structure

- **`document_pipeline.py`** - Complete document processing pipeline ⭐
- **`web_app.py`** - Lightweight Flask web interface
- **`system_check.py`** - System verification and debugging
- **`utils/`** - Utility modules
- **`data/`** - Documents and search index
- **`docs/`** - Complete documentation

## 📖 Documentation

**➡️ See `docs/README.md` for complete documentation, architecture, and troubleshooting.**

## 🎯 Features

- **Multi-format support** - PDF, TXT, MD, DOCX, images
- **Fast TinyLlama integration** - Lightweight and responsive
- **Vector search** - FAISS-powered document retrieval
- **Clean web interface** - Simple Flask-based UI
- **Robust error handling** - Comprehensive debugging tools
